JAVASCRIPT Homework

Lesson 23

  • Video 1 (Операторьі JAVASCRIPT)
    • Version 1 - false (оператор "+" не переводить тип даних)
    • Version 2 - true (оператор "*" переводить тип даних у number)
    • Version 3 - false (інкремент "++" працює тільки зі змінними)
    • Version 4 - false (у консоль виводиться попереднє значення змінної, до того, як почати вичисленння, тому що використовується інкремент у суфіксній формі)
    • Version 5 - false (найвищий приорітет у оператора "!", тому спочатку переводиться у протилежні значення з "!", потім застосовується оператор "&&", який має вищий приорітет за "||", а вже потім оператор "||" шукає перше true та повертає його значення)
    • Version 6 - true? (сумніваюся, бо з одного боку, відсутні ";" (але так можна, коли в один рядок), а з другого - оскільки перше значення не null/undefined, то повертається воно)
  • Video 2 (IF ELSE в JAVASCRIPT. Условньій оператор "?")
    • Version 1 - умова не виконується (число 1 не дорівнює жорстко рядку "1")
    • Version 2 - умова виконується (число 5 дорівнює рядку "1")
    • Version 3 - умова виконується (92 більше '11' (оператор ">" переводить тип даних), 58 менше 100, оператор "&&" повертає значення останнього true)
    • Version 4 - перша умова не виконується, друга умова виконується (0 повертає false, пробіл повертає true)
  • Video 3 (Цикльі FOR и WHILE в JAVASCRIPT. Директивьі break и continue)
    • Version 2 - false (невірно, тому що коли значення змінної стає "0", це вже false і код не виконується)
  • Video 5 (Строки в JAVASCRIPT. Тип даних string.)
    • Version 1 - false (невірно, тому що повинні бути зворотні лапки "``" для вставки довільного виразу)
    • Version 3 - false (невірно, тому що оператор "+" не переводить тип даних у number. Щоб отримати 579, треба перевести рядок "456" у тип даних number, наприклад, за допомогою унарного оператора "+")
    • Version 6 - false (невірно, тому що підрядок 'лан' починається з 3, починаючи з 4 його нема)
